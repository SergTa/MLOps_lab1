Предусловие: Распаковать все файлы из архива в один каталог
00. Установить docker мануал https://docs.docker.com/engine/install/ubuntu/
0. Добавить вашего пользователя в группу docker sudo usermod -aG docker имя_пользователя
1. запустить автоматическую установку sudo ./install.sh
2. После установки jenkins скопировать ключ из терминала и вставить его по адресу http://[локальный адрес машины]:[порт jenkins] в нашем случае 8080
3. Установить все плагины по умолчанию
